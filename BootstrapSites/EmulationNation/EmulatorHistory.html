This page describes the history of emulators.

Background Definitions

Emulator: hardware or software that permits programs written for one computer to be run on another computer

Emulation History

  • 1962: Emulation theory was developed by three engineers working at IBM.
  • 1963: The word, "emulator", was coined by IBM during the development of the NPL (IBM 360) product line, which used a new combination of software, microcode, and hardware.
  • 1965: The first emulator was written by Larry Moss, which ran programs from the IBM 7070 on the IBM System/360.
  • 1982: Intel released the Intel 80286, which could run its 8086 code.
  • 1989: Developers figured out how to dump the contents of a ROM chip into a file, and run the file through an emulator instead of needing the physical chip.
  • 1991: Video game system emulation emerged.
  • 1995: Emulation development exploded into a "scene" with ROMS dumped and distributed throughout the internet.
  • 1997: Companies began to create emulators of classic and modern machines for marketing purposes.
