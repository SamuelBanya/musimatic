Terminal Emulators

This page describes the terminal emulators, and the computers that they are modeled after.

Background Definitions

Terminal: combination of a keyboard and output device (such as a video display unit) by which data can be entered into or output from a computer or electronic communications system

Classic Computer Which Modern Terminal Emulators Are Modeled After

VT-100

Interesting Facts About The VT-100 Computer:

  • Released in 1978 by DEC (Digital Equipment Corporation)
  • One of the first terminals by DEC to feature the Intel 8080 microprocessor
  • Was based on the ANSI X3.64 standard, which controlled cursor location, color, and video options, and is still found in most terminal emulators which interpret some ANSI escape sequences as text.

Computer Terminal Types

Hard-Copy Terminals

Hard-Copy Terminal: Computer that is an electromechanical teletypewriter.

Intelligent Terminals

Intelligent Terminal: Computer that does its own processing, usually with a microprocessor.

Block-orientated Terminals

Block-orientated Terminal: Computer that communicates with its host in blocks of data, instead of communicating with a host one character at a time.
