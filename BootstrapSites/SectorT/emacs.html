History

Key Dates

  • 1972 OR 1974: Richard Stallman visited the Stanford AI Lab, and observed the lab's E editor which was written by Fred Wright, and was impressed by the editor's WYSIWG (What You See Is What You Get) behavior.
  • 1976: Emacs was originally written by Carl Mikkelsen, David A. Moon, and Guy L. Steele Jr. This incorporated many of the macros that were implemented by Richard Stallman.
  • 1979: Emacs was the main editor in MIT's AI lab and its Laboratory for Computer Science.
  • 1985: Richard Stallman released GNU Emacs, which was the free software implementation of the Emacs text editor.

Features Of Emacs

Emacs is a text editor by default, but be used to format and print documents when interfaced with external programs like LaTeX (prounounced "La-Tech"), Ghostscript, or a web browser. It is also able to display words, sentences, paragraphs, and source code. It also supports user-created keyboard macros by allowing users to edit the "emacs.el" file.

Useful Extensions For Emacs That Are Not Present In Other Editors

  • Org-Mode: This allows you to prepare documents, take notes, make TODO lists, and plan projects.
  • AUCTEX: This allows you to write and format TEX files in Emacs.
  • Anything: This allows you to easily jump between buffers in Emacs.
  • Magit: This is an interface to the Git Version Control System, which allows Emacs users to use Git commands directly inside Emacs.
